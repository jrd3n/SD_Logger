# SD Data logger

## design brief

1. log data to sd card with time
1. display live data
1. zero offset for sensors
1. set time
1. start recording

### ideas

1. one button could start recording, and if held for 3 seconds zero ofset
1. start recording is alway done to a new file.
1. use a LCD to display data.
